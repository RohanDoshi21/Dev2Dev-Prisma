## Install

```sh
npm install

# Prisma setup
npx prisma generate
npx prisma migrate dev
```

## Usage

```sh
npm run dev

# Prisma
npx prisma studio

# Postgres Docker Container
docker compose -f "docker-compose.only-db-test.yml" up -d --build
```
